SEO Commandments

  • Have relevant content on your page.
  • Use key words in your content.
  • Mobile, mobile, mobile.
  • Have accurate content.
  • Load as fast as possible.
  • Engage users as much as possible.
  • Accessibility is just as important as any other requirement.
  • As above, engage the user even when things go wrong.
  • Use proper meta tags, use Open Graph tags.
  • USE HTTPS! NO REASON NOT TO!
  • Use proper headings - structure your page logically. A single H1, a few H2s, various H3s, so on.
  • Use popups as little as possible. Don't annoy the user. DO NOT get in the way of content.

Error Classes

  1. 1xx - Website still operational
  2. 2xx - Inconsequential warnings, website operational
  3. 3xx - Redirected
  4. 4xx - Error related to user (client went to a page that doesn't exist)
  5. 5xx - Error related to server (server unable to serve pages i.e. ngnix errors)
